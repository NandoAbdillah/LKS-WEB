How to play game

  1. Input Username
  2. Choose level, ball, and country
  3. Player 1 move the character using the following buttons “A" to move left, “W” to jump, “D” to move right, “Space” to kick the ball
  4. Player 2 move the character using the following buttons “Left arrow” to move left, “Right arrow” to move right, “Up arrow” to jump,“Enter” to kick the bal
  5. The score will be increased if the player can score a goal against the opponent
